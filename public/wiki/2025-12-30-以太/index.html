ETHER

The Source of All Miracles

历史与发展
#

觉醒：符文时代
#

在历史的迷雾深处，远古时代的人们并不知晓“力量”的真谛。那是一个混沌而黑暗的纪元，各大古老种族凭借与生俱来的天赋，分割并统治着各个位面。

人，在那个时代，仅仅是强权者的食粮、奴隶与玩物。

直到一位自称为 “准” 的奴隶站了出来。

传说中，“准”只身前往奴隶主的宫殿，手中仅握着几枚刻有古怪 符文 的石块。那一日，夜空被极光般的绚烂光辉笼罩，恍若旧世界的消亡与新世界的诞生。

“准”将这些符文教授给了各个 族群。依靠着符文的力量，被奴役的族群终于拥有了反击的獠牙。最后，准带领众人统一了各界，并以前所未有的伟岸巨力分化世界，开辟上界，将他的族群接引而上，开启了全新的纪元。

后世尊称其为——准帝。

变革：方术时代
#

随着符文的普及，学者们逐渐触及了力量的本质。符文仅仅是媒介，其核心在于调动充斥于天地间、同时也流淌在每个生命体内的某种基础元素——人们将其命名为 以太 (Ether)。

符文虽强，却存在天然的局限性：篆刻繁琐、变通性差、晦涩难懂。

经过无数先贤的探索，一种能够直接与以太共鸣的语言被整理出来——以太原语 (Ether Primal Language)。

以太原语的原理： 不拘泥于发音形式，通过任意族群的语言进行“编译”，只要 意念（含义）与原语逻辑一致，辅以特定的 手势引导（动作）与高度集中的 精神力（专注），便能直接催动以太，通过 “方术” 演化出千变万化的 “道法神通”。

这一发现标志着笨重的符文时代落幕，灵活多变的 方术时代 正式兴起。

属性与传承
#

研究表明，人的潜力无穷,蕴含着独特的道法.而构成复杂道法的则为四种基础道法,也就对应着四大基础以太。

🔥 火 (Ignis)
💧 水 (Aqua)
⛰️ 地 (Terra)
🍃 风 (Ventus)

每个生命体天生便亲和其中的一种属性，这种亲和性往往受到族群血脉的影响。

  • 例： 显赫的 Nephele 家族，其后代几乎无一例外地展现出对火道的天赋。

道法的演进：从后天到先天
#

方术师们在四大基础道法之上，结合个人的感悟与经验，开发自己的潜力创造出了独属于自己的 “后天道法”。

当一种后天道法被推演至极致，施术者甚至能将这种法则刻印于世界本源之上，实现 “以后天逆证先天” 的壮举。

  • 效果： 道法不再局限于体内 以太，而是如同呼吸般自行运转，永恒存在。
  • 例： 陆吾神族的始祖曾将 宇道 刻入世界，使得其后代天生便拥有宇道的 先天天赋。

历史长河中，也有耀眼星辰,以其独具特色的道法神通绽放光芒：

  • 阳炎至尊：力道与阳道的极致。
  • 安方始祖：夜道的开创者。

综合评级规范
#

昭明及各界对方术师的实力划分有着严格的 “环” (Circle) 级标准。

环 称号 能力特征
1~5 方术师 掌握基础与进阶方术，探索个人潜力。
5~7 准王 自创五环方术，拥有独特的后天道法雏形。
7 圣人 自创七环方术，后天道法演化出多色神通，推陈出新。
8 王 / 主 / 魔王 / 神王 一方霸主。自创八环方术，低环方术瞬发随心。
代表：安方王、妖主、境龙王
9 帝 / 至尊 / 至高神王 / 圣约魔王 改变世界法则的力量。以后天逆证先天的传说境界。能够施展九环方术
代表：准帝、怜帝、阳炎至尊

施法媒介分类
#

现代方术体系主要依赖两大媒介流派。

ᚱ

符文媒介 (Rune)

  • 载体： 武器、魔杖、法器、躯体（血肉刻印）。
  • 优点： 零门槛，即拿即用，无需吟唱。
  • 缺点： 灵活性低，图案晦涩，破损即失效甚至发挥出相反的效果。
  • 场景： 适合高强度、快节奏的中近距离搏杀。
Ω

原语媒介 (Primal)

  • 载体： 书籍、符纸、大地、星空、心灵。
  • 优点： 上限极高，完美无损耗催动以太，效果多变。
  • 缺点： 需要专注，被打断会导致以太反冲（内伤）。
  • 场景： 适合阵地战、仪式、超远程打击。

现代革新：体系的融合
#

随着时代的进步，有两位伟大的至尊对方术体系进行了革命性的改良：

  1. 千道万法至尊：

    • 极大地压缩了原语的长度与限制。
    • 开发出 “扩展术式” 逻辑，实现了不同术式的叠加、增益与组合，让方术如正常交谈般灵活。

  2. 寰宇合一至尊：

    • 延续并统合了前者的思想，打破了符文与原语的壁垒。
    • 研发出 “神通发生器”：一种极其稳定的施法终端,仅需以太便可催动.
    • 建立了覆盖全界的 “道法网络”，实现了无障碍实时通讯监控与远程施法辅助。

正是古往今来无数求道者的薪火相传，才铸就了今日辉煌完备的方术系统。
